CSS Box Model
	Margin = area transparan di sekitar kotak( diluar border)
	Padding = area transparan di dalam kotak (antara content dan border)
		karateristik = gapunya nilai negatif, gapunya auto, padding merubah ukuran box 

	Border = batas disekeliling content dan padding 
		solid, dotted, dashed, double
		border: dotted;
	Content = konten sebenarnya didalam box, bisa berupa teks atau gambar

	Box sizing = auto mengatur jarak border
  
   div{
	width: 60px;
	margin: 10px auto;
	border-sizing: border-box;	}
